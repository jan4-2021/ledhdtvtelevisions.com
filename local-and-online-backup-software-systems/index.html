Comparing Local And Online Backup Software Systems

December 25th, 2010 by GuestPoster

Backup software has traditionally focused on copying files to a local device such as an external hard disk. However, some new backups services and software solutions can also backup your data by uploading is to an online server. In this article, we’ll go through where the benefits and setbacks of each approach that can be quite easily picked out.

Webmasters have used scripts to backup websites using duplicate servers for many years so it was only a matter of time before regular PC users could synchronize their data whith an online server. Commercial products such as the iDrive consist of a client installed application which can be set up to synchronize/upload data to their backup servers for safekeeping.

The client application is capable of automating the entire backup process once set up so that files that are changed or added are automatically being uploaded for safekeeping. Moreover, you don’t need any backup hardware to do the duplicates so this is a great option for backing up laptops while traveling. A major setback is the fact that you still need fast Internet connections for this to work properly. Moreover, you have to incur subscription costs on a monthly basis in order to join these types of services.

For many people, using backup software like SyncToy to clone data from their computer on to an external storage device/disk is still a popular option. The amount of data you can backup is only limited by the disk capacity (or number of blank CDs you have if you choose to backup locally by burning to disk). These approaches can also be very economical given the low cost of hard drives and blank media.

What you do have to weigh up however is the fact that this is a manual operation requiring your intervention to kick start the backups. As a result, there is an inclination to forget about doing backups and leave long gaps between making copies which can prove to be detrimental and result in data loss.
