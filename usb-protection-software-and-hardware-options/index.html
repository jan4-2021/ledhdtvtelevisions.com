USB Protection – Software And Hardware Options

September 5th, 2010 by GuestPoster

USB devices have made it possible to transport a large amount of personal data with you wherever you go. However, USB protection is a subject which few USB device users actually consider, to their regret if the device is stolen or lost and others can access the files. This article takes a look at the two main approaches to USB password protection by using software and hardware means.

The first approach to protecting the contents of a USB drive involves using software applications such as True Crypt and Bit Locker to Go. These applications can be installed on existing USB devices and give you the opportunity to password protect and encrypt the contents of the directories. This is an easy way of fixing computer security concerns regarding a USB drive and can also be incorporated with a company’s own backup software to synchronize data to a network server (when the device is inserted in a company PC).

However, there may be some concerns over the level of security if weak password is used by the users as the algorithm used to encrypt passwords may not suffice. That said, any encryption is better than none at all.

The second approach is to use a USB Flash device which has its own security chip installed. These products will automatically encrypt all data as it is being saved whether or not the user has chosen password are not. All documents are encrypted in memory so even if a hacker attempt to use data recovery software to read the Flash memory they will be unable to decrypt the contents.

The problem with having dedicated hardware security is that it will prove to be more expensive than simply going with the software approach. While companies like Lexar and Verbatim are already providing these types of devices, the prices have not come down sufficiently to make them competitive with their unsecured predecessors.

Each company should provide their own USB protection policy regarding the use and security of these devices within the workplace. If no policy is provided then you should consider both of the options presented here in order to secure your own information.
