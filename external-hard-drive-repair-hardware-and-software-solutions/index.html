External Hard Drive Repair – Hardware And Software Solutions

July 4th, 2011 by GuestPoster

Finding the right hardware and software related solutions when doing an external hard drive repair requires careful consideration. The major concern is that you could actually cause more data loss if you make the wrong type of repair.

Always start these types of checks by establishing if the hardware is not at fault. Start up your PC and connect the external hard drive then listen out for grinding or clicking sounds. This is an indication that the internal components have been damaged (e.g. the motor, spindle, platter, etc.). Shut down the PC and external hard drive and seek a computer technician to open and completes the repairs.

Next, go to the Device Manager utility and check for signs of faults with the external hard drive. It will identify if there are conflicts with other components on the system or problems with the driver software. PC freezes can often happen when there are conflicts with other USB devices (especially when using a splitter). The program also provides a troubleshooting utility which gives you specific suggestions related to the actual faults you’re seeing.

Do a visual inspection of the USB cable itself. Check that there is no problems with the pins and that they have not been pushed in or broken. You should also try using a different USB port in your PC as this may be the cause of the problem. Using a separate PC altogether is even more suggested as completely isolates the problem.

Finally, you have to check that the memory on the external hard drive has not been corrupted. Software for doing an external hard drive repair of bad sectors are relatively few and far between. Windows provides a disk checking utility which can prove to be quite effective [Windows Explorer/choose your external hard drive/properties/tools]. More specialist tools (e.g. HDD Regenerator) may be more effective at resolving bad sectors.
